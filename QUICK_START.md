# 🚀 快速开始指南

## 5分钟上手

### 第一步：安装依赖

打开终端，进入项目目录：

```bash
cd C:\Users\Administrator\surgery-scheduling-app
npm install
```

> 💡 **提示**: 首次安装可能需要 3-5 分钟，请耐心等待。

### 第二步：启动应用

```bash
npm start
```

看到以下信息表示启动成功：
```
** Angular Live Development Server is listening on localhost:4200 **
✔ Compiled successfully.
```

### 第三步：打开浏览器

访问: **http://localhost:4200**

🎉 恭喜！应用已经运行起来了！

## 🎯 首次使用

### 自动加载的模拟数据

应用首次启动会自动生成约 40-60 条模拟排班数据，包括：
- ✅ 近两周的手术排班
- ✅ 8种不同手术类型
- ✅ 6位医生的排班
- ✅ 5个手术室的使用
- ✅ 4种状态分布

### 快速体验核心功能

#### 1️⃣ 查看排班（10秒）
- 日历会显示所有排班
- 不同颜色代表不同状态
- 🔵 蓝色 = 已安排
- 🟢 绿色 = 进行中
- ⚫ 灰色 = 已完成
- 🔴 红色 = 已取消

#### 2️⃣ 创建新排班（1分钟）
1. 点击右上角 **"新建排班"** 按钮
2. 填写表单：
   - 患者姓名: 张三
   - 手术类型: 阑尾切除术
   - 主刀医生: 李医生
   - 手术室: 手术室1
   - 选择明天的某个时间段
   - 状态: 已安排
3. 点击 **"创建"** 按钮
4. ✅ 成功！可以在日历上看到新排班

#### 3️⃣ 拖拽调整时间（30秒）
1. 在日历上找到刚创建的排班
2. 按住鼠标左键拖拽到其他时间
3. 松开鼠标
4. ✅ 时间已更新！

#### 4️⃣ 使用筛选（30秒）
1. 在顶部筛选器中
2. 选择一位医生，如"李医生"
3. 自动显示该医生的所有排班
4. 点击"重置"恢复显示全部

#### 5️⃣ 切换视图（10秒）
1. 点击工具栏的 **"列表视图"** 按钮
2. 以表格形式查看所有排班
3. 点击 **"日历视图"** 切换回日历

## 📚 常用操作

### 创建排班的3种方式

**方式1: 按钮创建**
```
点击 "新建排班" → 填写表单 → 提交
```

**方式2: 双击创建**
```
在日历空白处双击 → 自动打开表单 → 填写并提交
```

**方式3: 选择时间创建**
```
在日历上拖选时间段 → 自动打开表单（时间已预填） → 填写其他信息并提交
```

### 编辑排班

**方式1: 日历点击**
```
点击日历上的排班卡片 → 编辑表单 → 修改 → 保存
```

**方式2: 列表编辑**
```
切换到列表视图 → 点击"编辑"按钮 → 修改 → 保存
```

### 删除排班

**方式1: 表单删除**
```
打开编辑表单 → 点击"删除"按钮 → 确认
```

**方式2: 列表删除**
```
列表视图 → 点击"删除"按钮 → 确认
```

### 筛选数据

```
日期范围: 选择开始和结束日期
医生: 下拉选择医生
手术室: 下拉选择手术室
状态: 多选状态（可选多个）
手术类型: 下拉选择类型

→ 自动应用筛选（300ms后）
```

## ❓ 常见问题

### Q: 数据保存在哪里？
A: 数据保存在浏览器的 localStorage 中，刷新页面不会丢失。

### Q: 如何清空所有数据？
A: 打开浏览器控制台（F12），执行：
```javascript
localStorage.removeItem('surgery_schedules');
location.reload();
```

### Q: 为什么无法拖拽？
A: 确保：
1. 在日历视图（不是列表视图）
2. 鼠标点击在排班卡片上
3. 目标位置没有冲突

### Q: 时间冲突怎么办？
A: 系统会自动检测冲突并提示。解决方法：
- 选择其他手术室
- 调整时间段
- 编辑或删除冲突的排班

### Q: 手机上如何使用？
A: 完全支持！打开手机浏览器访问相同地址即可。界面会自动适配。
